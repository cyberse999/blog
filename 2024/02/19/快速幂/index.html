例题

问题描述：如果一个整数序列 {1,2,...,n}\{1,2,...,n\}{1,2,...,n} 满足如下两个规则，我们可认为序列是美丽的：

规则 1：a1,...,aia_1,...,a_ia1​,...,ai​ 单调递减或单调递增的

规则 2：ai,...,ana_i,...,a_nai​,...,an​ 单调递减或单调递增的

现在给你一个序列 {1,2,...,n}\{1,2,...,n\}{1,2,...,n} ，请你求共有多少种排列情况可以使得该序列是美丽的。答案可能非常大，所以答案要模 p。

思路

考虑以下四种情况:

  1. 规则 111 和 222 都是单调递减，只有{n,n−1,...,1}\{n,n-1,...,1\}{n,n−1,...,1}满足条件
  2. 规则 111 和 222 都是单调递减，只有{1,2,...,n}\{1,2,...,n\}{1,2,...,n} 满足条件
  3. 规则 111 是单调递减，规则 222 是单调递增，此时必然有 ai=1a_i=1ai​=1，且在选定左边 i−1i-1i−1 个数的取值后其排列方式固定，右边同理。情况数为 2n−1−22^{n-1}-22n−1−2
  4. 规则 111 是单调递增，规则 222 是单调递减，此时必然有 ai=na^i=nai=n ，情况数为2n−1−22^{n-1}-22n−1−2

四种情况不存在重合，情况数可以直接相加，最终答案 2n−22^n-22n−2 ，直接快速幂计算即可。 注意此时 p⩽1018p\leqslant10^{18}p⩽1018 ，快速幂中间过程要使用 int128int128int128 。
