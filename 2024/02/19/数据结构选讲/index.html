T1 Balanced Array

称一个长度为 lll 的数列 aaa 为平衡的，当且仅当其满足下列条件

  • 存在一个整数 kkk，满足 1≤k≤l−121\le k\le \frac {l-1}21≤k≤2l−1​

  • 对于每个 i=1,2,...,l−2ki=1,2,...,l-2ki=1,2,...,l−2k，有 ai+ai+2k=2ai+ka_i+a_{i+2k}=2a_{i+k}ai​+ai+2k​=2ai+k​

现在给定一个长度为 nnn 的数列 aaa，请对于每个 i∈[1,n]i\in[1,n]i∈[1,n]，判断长度为 iii 的前缀 a1,a2,...,aia_1,a_2,...,a_ia1​,a2​,...,ai​ 组成的数列是否为平衡的。

数据范围：n≤2×106,1≤ai≤2×108n\le 2\times 10^6,1\le a_i\le 2\times 10^8n≤2×106,1≤ai​≤2×108

题解

题目的限制条件现相当于对于每个 i∈[1,k]i\in[1,k]i∈[1,k]，ai,ai+k,ai+2k,...a_i,a_{i+k},a_{i+2k},...ai​,ai+k​,ai+2k​,... 是一个等差数列，每个 iii 对应的等差数列的公差可能不一样。

考虑对于一个 kkk，存在 2k+1≤l<r2k+1\le l<r2k+1≤l<r，且 [1,r][1,r][1,r] 对应的数列在当前 kkk 下能够满足题目的限制条件，那么显然由于 [1,l][1,l][1,l] 的数列更短且被 [1,r][1,r][1,r] 所包含，所以 [1,l][1,l][1,l] 对应的数列在当前 kkk 下也能够满足题目的限制条件。

换言之，每个 kkk 对应的合法区间是从 2k+12k+12k+1 开始的连续一段，如果找到了这个区间，那么可以通过差分和前缀和轻松维护最终的答案。

对于每个 kkk 而言，满足条件的一段前缀一定满足 [ak+1,...,a2k]−[a1,...,ak]=[a2k+1,...,a3k]−[ak+1,...,a2k]=...=[d1,...dk][a_{k+1},...,a_{2k}]-[a_1,...,a_k]=[a_{2k+1},...,a_{3k}]-[a_{k+1},...,a_{2k}]=...=[d_1,...d_k][ak+1​,...,a2k​]−[a1​,...,ak​]=[a2k+1​,...,a3k​]−[ak+1​,...,a2k​]=...=[d1​,...dk​]，其中这里的减号代表每一个位置的数分别相减。

如果可以快速求解两段长度相同的序列相减后得到的新序列并且可以快速比较新序列是否相同就可以对每个 kkk 找到最长的满足条件的序列。

考虑采用类似哈希的方式，记一个长度为 kkk 的序列 b1,b2,...,bkb_1,b_2,...,b_kb1​,b2​,...,bk​ 的哈希值为 b1×M0+b2×M1+...+bk×Mk−1b_1\times M^0+b_2\times M^1+...+b_k\times M^{k-1}b1​×M0+b2​×M1+...+bk​×Mk−1，那么将另一个长度也为 kkk 的序列 {c}\{c\}{c} 的哈希值与其相减，得到的结果为 M0(b1−c1)+M1(b2−c2)+...+Mk−1(bk−ck)M^0(b_1-c_1)+M^1(b_2-c_2)+...+M^{k-1}(b_k-c_k)M0(b1​−c1​)+M1(b2​−c2​)+...+Mk−1(bk​−ck​)

对于[ak+1,...,a2k]−[a1,...,ak],[a2k+1,...,a3k]−[ak+1,...,a2k][a_{k+1},...,a_{2k}]-[a_1,...,a_k],[a_{2k+1},...,a_{3k}]-[a_{k+1},...,a_{2k}][ak+1​,...,a2k​]−[a1​,...,ak​],[a2k+1​,...,a3k​]−[ak+1​,...,a2k​]，如果它们的哈希值相减是相同的，那么就代表对应位置可以成为一个等差数列，也就意味着 [1,3k][1,3k][1,3k] 这个前缀是满足题意的。

类似的可以判断 [1,4k],[1,5k],...,[1,n][1,4k],[1,5k],...,[1,n][1,4k],[1,5k],...,[1,n] 的这些前缀是否满足题意，直到判断到某一个长度为 kkk 的整数倍的前缀不满足题意，就在其中二分找最后的一个满足题意的位置。

时间复杂度 O(n+n/2+n/3+...)=O(nlog⁡n)O(n+n/2+n/3+...)=O(n\log n)O(n+n/2+n/3+...)=O(nlogn)。

T2 Barkley II

给定一个长度为 nnn 的数列 aaa，请找到一个连续区间 al,al+1,...,ara_l,a_{l+1},...,a_ral​,al+1​,...,ar​ 使得其中不同的数字个数减去其中数字的mex最大，输出这个最大值。

  • 一个数列的mex是其中没有出现过的最小正整数。

数据范围：多组数据，∑n≤5×105,1≤ai≤5×105\sum n\le 5\times 10^5,1\le a_i\le 5\times10^5∑n≤5×105,1≤ai​≤5×105

题解

考虑一个序列 {a}\{a\}{a} 中没有出现 xxx，那么它的 mex 至多为 xxx。

所以对于一个数 xxx，只需要枚举不包含它的极长连续子序列，并假定这个序列的 mex 是 xxx。统计其中出现的数字的种类数，再减去 xxx 的值就是当前子序列的答案。

上述情况必定能得出最优解，因为对于不包含 xxx 的极长连续子序列，如果其 mex 不为 xxx，那么在枚举到其 mex 时就可以获得当前区间对应的最优解。

mex 的枚举范围只需要从 111 枚举到第一个没出现过的数字即可，其它数字必然不可能成为 mex。

考虑 mmm 个相同的数字会将序列划分为至多 m+1m+1m+1 段极长连续子序列，最多只需要统计 2n2n2n 个区间中不同数字的数目，可以采用莫队，但是有超时的风险。

因为统计的是不同的数字，可以认为某个区间中所有相同的数字之中的最后一个才对答案有贡献。

于是将所有需要统计的区间按照右端点进行排序，每次移动右端点加入新数字时只需要把它当成目前的最后一个，将它的权值改为 111，并将本来权值是 111 的倒数第二个相同数字的权值改为 000，这样 [l,r][l,r][l,r] 间的权值和就是 [l,r][l,r][l,r] 区间的不同数字个数。

转化为单点加+区间求和问题，利用树状数组即可在 O(nlog⁡n)O(n\log n)O(nlogn) 的时间复杂度内求解。

T3 Rainbow Subarray

对于一个长度为 nnn 的数列 aaa，我们认为连续子序列 al,...,ara_l,...,a_ral​,...,ar​ 是一个"彩虹子序列"当且仅当对于所有 l≤i<rl\le i<rl≤i<r，有 ai+1−ai=1a_{i+1}-a_i=1ai+1​−ai​=1。

你可以进行 kkk 次操作，每次可以将数列里的某个元素增加或减少 111。请最大化在进行至多 kkk 次操作后数列里最长的彩虹子序列的长度，并输出这个长度。

数据范围：多组数据，∑n≤5×105,0≤k≤1015,1≤ai≤109\sum n\le 5\times 10^5,0\le k\le 10^{15},1\le a_i\le 10^9∑n≤5×105,0≤k≤1015,1≤ai​≤109

题解

[l,r][l,r][l,r] 间的连续子序列中 ai+1−ai=1a_{i+1}-a_i=1ai+1​−ai​=1 等价于对于所有 i,j∈[l,r]i,j\in [l,r]i,j∈[l,r] 有 ai−i=aj−ja_i-i=a_j-jai​−i=aj​−j，于是将每个数减去它的下标，问题转化为在至多 kkk 次加减操作后求最长的相等子序列。

  • 将一个连续区间的所有数变为相等的最优解是全部变成中位数

如果一个区间 [l,r][l,r][l,r] 可以在至多 kkk 次操作内变为全部相等，那么对于区间 [l1,r1]⊆[l,r][l_1,r_1]\subseteq [l,r][l1​,r1​]⊆[l,r] 肯定也可以在 kkk 次操作内变为全部相等，于是可以采用双指针来对于每个右端点 rrr 找到最远的 lll 满足 kkk 次操作之内可以将区间内的所有数变成相等的。

首先需要动态维护滑动窗口的中位数，可以采用两个 set，第一个 set S1S_1S1​ 放区间内最小的一半数，第二个 set S2S_2S2​ 放区间内最大的一半数，如果在加入或者删除一个数后两个 set 的大小之差大于 111，那么就从大小较大的那个 set 中把对应的数移动到另一个 set 中，最后如果 ∣S1∣≥∣S2∣|S_1|\ge|S_2|∣S1​∣≥∣S2​∣，那么 S1S_1S1​ 中最大的那个数就是中位数。

假设中位数是 mmm，最终每个区间变成相同需要的操作数就是 m∣S1∣−sum1+sum2−m∣S2∣m|S_1|-sum_1+sum_2-m|S_2|m∣S1​∣−sum1​+sum2​−m∣S2​∣，其中 sum1,sum2sum_1,sum_2sum1​,sum2​ 代表 S1,S2S_1,S_2S1​,S2​ 中的元素和，在修改其中元素时动态维护即可。

T4 Restore Atlantis

给定 nnn 个平面直角坐标系上的矩形，每一个矩形左上角坐标为 (xai,yai)(xa_i,ya_i)(xai​,yai​)，右下角坐标为 (xbi,ybi)(xb_i,yb_i)(xbi​,ybi​)，矩形之间可能有重合部分。

有 qqq 个询问，每个询问给定 si,tis_i,t_isi​,ti​，你需要求出在忽略第 sis_isi​ 到第 tit_iti​ 个输入的矩形时剩余的矩形覆盖的总面积，矩形之间的重合部分在计算总面积时只考虑一次。

数据范围：1≤n,q≤105,0≤xai,xbi,yai,ybi≤20001\le n,q\le 10^5,0\le xa_i,xb_i,ya_i,yb_i\le 20001≤n,q≤105,0≤xai​,xbi​,yai​,ybi​≤2000

题解

对于每个格子 (i,j)(i,j)(i,j)，如果知道最早覆盖它的矩形是第 min[i][j]min[i][j]min[i][j] 个，最晚覆盖它的矩形是第 max[i][j]max[i][j]max[i][j] 个，那么这个格子在某个询问中没有被覆盖的条件就是 si≤min[i][j]≤max[i][j]≤tis_i\le min[i][j]\le max[i][j]\le t_isi​≤min[i][j]≤max[i][j]≤ti​。

考虑利用扫描线逐行确定每个格子的 min,maxmin,maxmin,max。将矩形分解为 (i,yai,xai,xbi,Add)(i,ya_i,xa_i,xb_i,Add)(i,yai​,xai​,xbi​,Add) 与 (i,ybi+1,xai,xbi,Del)(i,yb_i+1,xa_i,xb_i,Del)(i,ybi​+1,xai​,xbi​,Del)，再建立一棵线段树，线段树的叶子对应这一行的每个格子，对线段树的每个区间 [l,r][l,r][l,r] 用两个优先队列维护覆盖当前行标号为 [l,r][l,r][l,r] 的格子的矩形的最早和最晚编号。

对于Add操作，只需要在下放的过程中在对应线段树区间的优先队列中把矩形的编号加入即可。

对于Del操作，可以采用懒删除的方式，在每个区间中再用两个优先队列用相同方式维护删除的标号，只要维护 max,minmax,minmax,min 的优先队列队首不是需要删除的标号就可以不用去执行删除操作。

处理完每一行的操作后对线段树 dfs 到每个叶子节点，同时维护涉及到的区间的所有优先队列队首的值取 max 和 min 的值，到叶子节点的取值就是当前格子的 max,minmax,minmax,min。

假设行数和列数为 CCC，预处理时间复杂度为 O(nlog⁡nlog⁡C+C2)O(n\log n\log C+C^2)O(nlognlogC+C2)

统计出考虑所有矩形的条件下被覆盖的格子个数，对于每个询问，原本被覆盖而在询问的条件下没有被覆盖的格子满足 si≤min[i][j]≤max[i][j]≤tis_i\le min[i][j]\le max[i][j]\le t_isi​≤min[i][j]≤max[i][j]≤ti​。

可以将所有格子对 max[i][j]max[i][j]max[i][j] ，所有询问按照 tit_iti​ 排序，并统计每个询问中相对于不删除矩形的情况下减少的格子数量。

这些格子满足 max[i][j]≤timax[i][j]\le t_imax[i][j]≤ti​，在排序后可以用双指针的方式加入满足这个条件的格子，同时将 min[i][j]min[i][j]min[i][j] 位置的权值+1，那么 [si,ti][s_i,t_i][si​,ti​] 之间的权值和就是 si≤min[i][j]≤max[i][j]≤tis_i\le min[i][j]\le max[i][j]\le t_isi​≤min[i][j]≤max[i][j]≤ti​ 的格子个数，用总数减去这个数量就是这个询问的答案。

单点加+区间求和，使用树状数组即可在 O(qlog⁡n+C2log⁡C)O(q\log n+C^2\log C)O(qlogn+C2logC) 的复杂度内完成求解。

T5 Trapping Rain Water

给定一个长为 nnn 的序列 a1,a2,...,ana_1,a_2,...,a_na1​,a2​,...,an​，共 qqq 次操作，第 iii 次操作会将 axia_{x_i}axi​​ 的值增加 viv_ivi​，设 fif_ifi​ 表示前缀最大值，gig_igi​ 表示后缀最大值，每次操作后求 $$\sum_{i=1}^n(\min(f_i,g_i)-a_i)$$

数据范围：多组数据，1≤n,q≤105,1≤ai,vi≤106,1≤∑n,∑q≤1061\le n,q\le 10^5,1\le a_i,v_i\le 10^6,1\le \sum n,\sum q\le 10^61≤n,q≤105,1≤ai​,vi​≤106,1≤∑n,∑q≤106

题解

假设 aj=max⁡{ai}a_j=\max\{a_i\}aj​=max{ai​}，对于 fif_ifi​ 而言，在 i≥ji\ge ji≥j 后 fi=ajf_i=a_jfi​=aj​ 并保持不变，对于 gig_igi​ 而言，在 i≤ji\le ji≤j 后 fi=ajf_i=a_jfi​=aj​ 并保持不变。

所以对于一个位置 iii，必然有 max⁡{fi,gi}=aj\max\{f_i,g_i\}=a_jmax{fi​,gi​}=aj​，min⁡{fi,gi}=fi+gi−aj\min\{f_i,g_i\}=f_i+g_i-a_jmin{fi​,gi​}=fi​+gi​−aj​

所以 ∑i=1nmin⁡{fi,gi}−ai=∑i=1nfi+∑i=1ngi−nmax⁡{ai}−∑i=1nai\sum_{i=1}^n\min\{f_i,g_i\}-a_i=\sum_{i=1}^n f_i+\sum_{i=1}^n g_i-n\max\{a_i\}-\sum_{i=1}^n a_i∑i=1n​min{fi​,gi​}−ai​=∑i=1n​fi​+∑i=1n​gi​−nmax{ai​}−∑i=1n​ai​

max⁡{ai}\max\{a_i\}max{ai​} 与 ∑ai\sum a_i∑ai​ 都好维护，问题来到了如何维护 fi,gif_i,g_ifi​,gi​。

对于一个固定的序列 aaa，fif_ifi​ 是逐段单调递增的，可以维护序列 aaa 的一个单调栈，栈内的每个元素控制的区间长度乘以这个元素的值就是 ∑fi\sum f_i∑fi​。

考虑修改操作，可以用 set 去维护这个单调栈，set 里面维护一个 pair 代表单调栈里面每个元素在原序列里面的位置和这个元素的值。修改操作就可以找到这个元素所在的区间，看这个元素加入 set 以后是否会改变单调栈的内容，如果会则将 set 的当前区间分裂为两个区间，并且看单调栈后面的元素是否比这个元素更小，如果是就还需要删除后面的元素。

gig_igi​ 的维护相当于将 aaa 序列倒过来用相同的方式维护。

每个元素在初始和修改的时候分别最多只会进入 set 一次，所以复杂度为 O((n+q)log⁡(n+q))O((n+q)\log (n+q))O((n+q)log(n+q))
